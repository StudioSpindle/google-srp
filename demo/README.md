# Demo

Install dependencies (make sure you're in the demo folder)

```bash
$ cd demo
```
```bash
$ npm i
```

Create a .env file in the root of the demo folder with your credentials:

**./demo/.env**
```html
GOOGLE_API_KEY="your-API-key-here"
GOOGLE_ENGINE="your-engine-key-here"
```

Then run:
```bash
$ npm run serve
```
